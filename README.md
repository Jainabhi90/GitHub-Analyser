# GitHub Analyser 🐙

A developer analytics dashboard built with React and Chart.js.

## Features
- Search any GitHub username
- View profile, followers, repos
- Language distribution pie chart
- Top 5 repos bar chart
- Search history
- Responsive design

## Tech Stack
- React + Vite
- Tailwind CSS
- Chart.js
- GitHub REST API
- Deployed on Vercel
